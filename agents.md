# Antigravity Persistent Instructions

## Core Principles

1. **i18n First**: Always consider translation files when developing or updating UI.
2. **Default Language**: Initial language should always be **English (EN)** for new users or when no preference is saved.
3. **Supported Languages**:
   - EN (English)
   - ES (Español)
   - FR (Français)
   - PT (Português)

## UI Guidelines

- Use minimalist, technical badges for language selectors (ISO codes only).
- Avoid flags or emojis for language representation.
- Maintain a "Vibe Coding" aesthetic: sober, technical, and efficient.
