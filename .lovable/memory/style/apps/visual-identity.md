# Memory: style/apps/visual-identity
Updated: now

Los badges de verificación de aplicaciones utilizan un esquema de color invertido (fondo primario con texto claro). El estado "Sin Verificar" usa un fondo rojo mate elegante (rose-100/rose-700). Los estados de las aplicaciones emplean una paleta premium:
- **Live/Active**: Verde esmeralda (emerald-50/emerald-700)
- **Building**: Turquesa mate (teal-50/teal-700)
- **Otros**: Gris neutro (gray-100/gray-600)

Colores centralizados en `src/lib/appStatusColors.ts` para consistencia en todos los componentes.
