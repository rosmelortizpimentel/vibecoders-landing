# Memory: features/beta-squads/diseno-pestana-beta
Updated: now

La pestaña de gestión 'Beta Squad' en el dashboard (/me/beta) utiliza un diseño de master-detail responsive:
- **Desktop**: Grid de dos columnas con lista de apps a la izquierda (col-span-4/3) y panel de configuración a la derecha (col-span-8/9)
- **Mobile**: Vista de lista inicial con transición a detalle al seleccionar una app, con botón de retroceso

Cada tarjeta de app en la lista muestra logo, nombre, tagline y un Switch para activar/desactivar el programa beta directamente. La selección visual usa borde primario y fondo con opacidad.
